Combine video clips - The GoPros split up the audio files after 15 minutes, this combines those split clips together.

Media to sync start time - This uses the text file and the audio start time to set the start time for all of the files to the same time. i.e Audio start = 10:23:30, Forward Video = 10:23:20, Front Video = 10:23:25. The audio is the last time, so the other two will be trimmed to start at 10:23:30.

Replace audio from video - As it sounds, the go pro audio is terrible and if you want to watch an event you can replace the gopro audio with the audio from the microphone so you dont have to manually do it yourself in a program.
